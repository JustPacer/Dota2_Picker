# 🛡️ Dota 2 Draft Analyzer

Интеллектуальный анализатор драфтов для Dota 2, разработанный на **Python**, **Streamlit** и **SQLite** с использованием исторических данных из **OpenDota API**.

---

## 🚀 Основной функционал

1. **Анализ лайнинга (до 10-й минуты)**
   - Прогнозирует исход линий на основе исторического превосходства по нетвордсу (Net Worth) и опыту (XP) к 10-й минуте матча.
   - Сравнивает силы героев на конкретных линиях (Сейфлейн vs Офлейн, Мид vs Мид).

2. **Прогноз винрейта пика (Draft Win Probability)**
   - Рассчитывает общую вероятность победы выбранной комбинации из 5 героев Radiant против 5 героев Dire.
   - Формула учитывает базовую силу героев (Solo Win Rate), личные матчапы (контрпики) и синергию внутри команды, сглаженные методом **аддитивного байесовского сглаживания (Bayesian Smoothing)** для фильтрации шума при малом количестве игр.

3. **Подсветка синергии пар (Дуэты)**
   - Находит связки внутри одной команды, совместный винрейт которых значимо выше их сольного винрейта, и подсвечивает их в интерфейсе.

4. **Автоматический Рол-Фиксер (Role Fixer)**
   - При выборе героев автоматически распределяет их по позициям (с 1 по 5) на основе мете патча 7.41d (через алгоритм взвешенного двудольного сопоставления).
   - Поддерживает ручное изменение ролей в интерфейсе с помощью автоматического **свапа (обмена)** позициями между героями.

5. **--Умные рекомендации («Suggest Best Pick»)--**
   - Помогает при неполном драфте: подбирает топ-5 лучших персонажей на пустующую роль.
   - Расчет рекомендаций строится на поиске оптимальных синергий с союзниками и контрпиков против оппонентов с жестким ролевым ограничением (кандидат тестируется строго в выбранной позиции).

## 📂 Структура проекта

- `app.py` — веб-интерфейс приложения на Streamlit.
- `draft_analyzer.py` — математические алгоритмы анализа драфта, распределения ролей и подбора рекомендаций.
- `database.py` — интерфейс для работы с локальной базой SQLite.
- `load_data.py` — парсер OpenDota API, создающий и наполняющий локальную базу кэша.
- `test_analyzer.py` — автоматические тесты для проверки корректности математических моделей и ролевого сопоставления.
- `Dota2DraftAnalyzer.vbs` — VBS-скрипт скрытого фонового запуска сервера и открытия браузера по двойному клику.
- `StopAnalyzer.vbs` — VBS-скрипт для быстрой остановки фонового сервера и очистки процессов.

## 🛠️ Установка и запуск

### Шаг 1. Установка зависимостей
Убедитесь, что у вас установлен Python (версии 3.8+). Установите библиотеку Streamlit:
```bash
pip install streamlit
```

### Шаг 2. Импорт и кэширование данных
Запустите скрипт для загрузки актуальных данных матчей патча 7.41d с OpenDota Explorer и создания локальной базы данных:
```bash
python load_data.py
```
*База данных `opendota_cache.db` будет создана локально. Это нужно сделать один раз.*

### Шаг 3. Запуск приложения

#### Вариант A. Стандартный запуск через терминал:
```bash
streamlit run app.py
```
После запуска приложение откроется по адресу `http://localhost:8501`.

#### Вариант B. Удобный запуск в 1 клик (без терминала):
1. Дважды кликните по файлу **`Dota2DraftAnalyzer.vbs`** в папке проекта. Сервер запустится скрытно, и приложение само откроется в вашем браузере.
2. Когда закончите работу, дважды кликните по файлу **`StopAnalyzer.vbs`**, чтобы закрыть фоновые процессы сервера.
